Virtual Audio Driver files go here.

Download the signed release from:
https://github.com/VirtualDrivers/Virtual-Audio-Driver/releases

Extract the ZIP and place these files in this directory:
  - VirtualAudioDriver.inf
  - VirtualAudioDriver.sys
  - virtualaudiodriver.cat

These files are bundled into the SoundDome installer and installed
via pnputil during setup.
